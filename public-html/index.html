balenalabs-browser

A balena block for browsing HTML

Choosing what to display

You have a few options for choosing what to display here:

  • You can create a service on this device serving HTTP over port 80 or 8080, or HTTPS over 443, and it'll display here automatically! **
  • You can set a Device Variable called LAUNCH_URL
  • You can use the API on port 5011 to set the URL (see project readme for details)
** The balenalabs-browser service will need network_mode: host set.
